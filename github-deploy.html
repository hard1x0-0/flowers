Your Rose Day app will be live at: yourname.github.io/flowers

Step 1: Create GitHub Repository

Go to GitHub and create a new repository:

‚ûú Create New Repository

Fill in the details:

  • Repository name: flowers
  • Description: Interactive Rose Day Flower Drawing App
  • Visibility: Public
  • Leave all other options as default

Then click "Create repository"

Step 2: Copy Your Repository URL

After creating the repo, you'll see a page with a green "Code" button.

Click it and copy the HTTPS URL, it should look like:

https://github.com/YOUR_USERNAME/flowers.git

üí° Keep this copied for the next step!

Step 3: Push Your Code to GitHub

Execute these commands in PowerShell (one at a time):

Command 1: Set the remote URL

git remote set-url origin https://github.com/YOUR_USERNAME/flowers.git

Replace YOUR_USERNAME with your GitHub username

Command 2: Push to GitHub

git push -u origin main

It will ask for your GitHub username and password (use personal access token if you have 2FA)

Step 4: Enable GitHub Pages

Go back to your repository on GitHub and:

  1. Click "Settings" tab at the top
  2. Scroll down to "Pages" on the left menu
  3. Under "Source", select "Deploy from a branch"
  4. Select "main" branch and "/ (root)" folder
  5. Click "Save"

‚úÖ Wait 1-2 minutes, then your site will be live!
